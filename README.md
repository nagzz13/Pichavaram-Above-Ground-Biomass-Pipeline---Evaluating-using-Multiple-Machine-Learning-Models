# Pichavaram-Above-Ground-Biomass-Pipeline---Evaluating-using-Multiple-Machine-Learning-Models
End-to-end pipeline for mapping mangrove above-ground biomass in Pichavaram — fuses CubeSat optical indices, Sentinel-1 SAR and TanDEM-X canopy height, trains RF/XGBoost/CatBoost/KNN with SHAP feature selection, and produces a weighted-ensemble AGB map with per-pixel uncertainty.
